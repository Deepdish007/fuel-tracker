# ⚡ Fuel — Calorie Tracker

A mobile-first calorie and macro tracking app. Works entirely in your browser with no backend needed — data is saved locally on your device.

## Deploy to GitHub Pages (2 minutes)

1. **Create a new repository** on GitHub (e.g. `fuel-tracker`)
2. **Upload `index.html`** into the root of the repo
3. Go to **Settings → Pages**
4. Under *Source*, select **Deploy from a branch → main → / (root)**
5. Click **Save** — your app will be live at:
   `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME`

> Tip: Bookmark the URL on your phone's home screen:
> - **iOS**: Share → Add to Home Screen
> - **Android**: Chrome menu → Add to Home Screen

## Features

- **Today** — log food with name, calories, protein, carbs & fat. Visual calorie ring + remaining calories display.
- **Saved Meals** — create reusable meal templates (Breakfast, Post-Workout, etc.) added in one tap.
- **History** — 7-day bar chart, full daily log, weekly & monthly calorie averages vs your goal, weekly budget calculator.
- **Settings** — daily calorie & macro goals. Export/import data as JSON.

## Data Storage

All data lives in your browser's `localStorage` — no account or server needed. Use **Settings → Export as JSON** to back up or transfer to another device.
